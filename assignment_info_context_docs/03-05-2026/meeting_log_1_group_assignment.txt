This assignment is used to keep track of your master’s project meetings. All meeting notes must be maintained in your team’s GitHub project. In the submission text box, paste the direct link to the "/meetings/" folder in your team’s GitHub repository (or the equivalent location where your meeting notes are stored).

 

Meetings to track:

Advisor Meetings (Team + Project Advisor)
Meetings with all students in your project team and your project advisor
Minimum expectation: At least 1 advisor meeting every two weeks
Team-Only Meetings (Students Only)
Meetings with only the students in your project team
Minimum expectation: At least 1 team-only meeting every week
 

GitHub requirements:

Create a “/meetings/” folder in your GitHub project.
Use one note per meeting, for example:
YYYY-MM-DD-advisor.md
YYYY-MM-DD-team.md
Notes may be written in Markdown (.md) or text (.txt).
Include action items and ownership clearly.
These records will be monitored every two weeks.
